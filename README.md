# 🚀 ML Projects Scratch

A collection of Machine Learning projects and algorithm implementations built to understand the mathematics, intuition, and engineering behind modern ML.

Rather than treating machine learning as a black box, this repository documents my journey of implementing algorithms from first principles, experimenting with datasets, and exploring how they work under the hood.

---

## 🎯 Purpose

This repository serves as a central place for all my Machine Learning projects—from complete applications to algorithms implemented from scratch.

My focus is on:

- Understanding the mathematics behind ML
- Building intuition before using libraries
- Writing clean, well-documented implementations
- Experimenting with real-world datasets
- Tracking my learning journey publicly

# 📚 Learning Philosophy

I believe that understanding **why** an algorithm works is more valuable than simply knowing **how** to use it.

Whenever possible, I implement algorithms from scratch before relying on high-level libraries.

This repository is an ongoing record of that journey.
